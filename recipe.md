## Creating a new working project environment 
1. Go to Github and select a repository.
2. Select the green "<> Code" button, select Codespaces, and create a new Codespace.
3. Open VS Code on your local machine. 
4. Make sure the Codespaces extension is installed on VS Code. 
5. Select "Connect to" and connect to your new Codespace. 
6. Now, you can open add files into the repository locally on VS Code. 
7. Open a bash terminal. Create and activate your virtual environment.(python -m venv .venv, and source .venv/bin/activate)
8. Create a requirements.txt to include all packages used in project. 
9. Once you are ready to bring the changes back up to your Github repository, you can git add the files (all or selected ones).
