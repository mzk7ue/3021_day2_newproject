## Creating a new working project environment 
1. Go to Github and select a repository.
2. Select the dropdown menu on the the green "<> Code" button.
3. Click on the Codespaces tab and create a new Codespace.
4. Open VS Code on your local machine. 
5. Open the extensions menu (the four box icon on the left hand side) and install the Codespaces extension.
6. In the welcome page of VS Code, click "Connect to..."
7. Find "Connect to Codespace" and click on the new Codespace you made in Github.
8. Once everything loads, open a bash terminal.
9. Create a virtual environment (python -m venv .venv) and activate it (.venv/bin/activate)
10. Create a requirements.txt to include all the package versions used in the project.
11. File save all work. 
12. Once you are ready to bring the changes back up to your Github repository, you can push them (git add, git commit, and git push).
